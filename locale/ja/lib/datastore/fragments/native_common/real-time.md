
## リアルタイムでデータを観察する

モデルの変更を購読できます。 これは、基盤となるストレージエンジンへのデータの更新に動的に反応します。 これは、クラウドと同期している場合は、GraphQL サブスクリプションのほか、AppSync API のバッキングに対して実行されるクエリまたは変異の結果である可能性があります。
